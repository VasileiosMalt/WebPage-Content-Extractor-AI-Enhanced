# WebPage Content Extractor - AI-Enhanced

**Extract webpage content as TXT and get AI-powered summaries.**

This extension allows you to easily extract the main content of any webpage into a clean text file. It also integrates with various AI providers (OpenAI, Gemini, GitHub Models, etc.) to generate intelligent summaries and analyses of the content directly within your browser.

**How to Use:**
1.  Navigate to any article or webpage.
2.  Click the extension icon.
3.  Hit **"Extract as TXT File"** to save the raw text.
4.  (Optional) Enable **"AI Analysis"**, select your provider, enter your API key, and click **"Extract + AI Analysis"** to get a comprehensive summary saved alongside the text.
